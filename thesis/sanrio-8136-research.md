# Sanrio (8136.T) Research Report

**日期**：2026-05-19
**类型**：IP 运营公司深度研究 — 非投资建议，作为 Pop Mart thesis 的竞对/参照系研究

---

## Executive Summary

Sanrio 是全球最成功的角色 IP 授权公司，旗下 Hello Kitty 累计创造超 $80B 商品零售额。公司经历了教科书级的 "辉煌→衰落→重生" 周期：2014 年营业利润峰值 ¥21B → 2020 年跌至 ¥2.1B → 2021 年首次亏损 → 新 CEO 上任后 V 型反转 → FY2025 营业利润 ¥51.8B（历史新高） → FY2026 上修至 ¥75.1B。

核心商业模式是**纯 IP 授权**——无制造、无库存、无关税风险。1,700+ 授权商覆盖 109 个国家，每年 50,000+ SKU。这与 Pop Mart 的自有零售重资产模式形成鲜明对比。

## 1. 公司简史

### 1.1 创立与早期（1960-1974）

| 时间 | 事件 |
|------|------|
| 1960 | 辻信太郎以 ¥100万资本创立"山梨丝绸公司" |
| 1962 | 发现在橡胶凉鞋上加可爱图案能显著提升销量——萌生"角色商品化"理念 |
| 1973 | 正式更名 **Sanrio**，推出首个原创角色 Coro Chan |
| 1974 | **Hello Kitty 诞生**——设计师清水侑子创作，灵感来自父亲送的小猫 |
| 1975 | Hello Kitty 首批商品上市（零钱包，售价 ¥240） |

### 1.2 Hello Kitty 出海与全球扩张（1976-2000）

| 时间 | 事件 |
|------|------|
| 1976 | **首家美国门店**开业（San Jose, Eastridge Mall），Sanrio Inc. 美国总部设于 San Jose |
| 1976 | 开始对外授权 Hello Kitty 形象——授权模式起步 |
| 1984 | 与 UNICEF 建立合作关系，品牌公益化 |
| 1980s 末 | Hello Kitty and Friends 动画播出，英文配音进入美加家庭 |
| 1990 | **Sanrio Puroland** 主题乐园开业（东京多摩市） |
| 1990s | Hello Kitty 从儿童市场破圈进入成人消费，成为日本文化 icon |
| 1996 | 年度角色人气投票（Character Ranking）开始举办 |

### 1.3 全盛期（2000-2014）

| 时间 | 事件 |
|------|------|
| 2000s | Hello Kitty 全球授权网络成熟，覆盖 130+ 国家 |
| 2005 | 聘请前 Disney 高管 Mas Imai 出任 Sanrio Inc. 总裁，推动美国市场专业化运营 |
| 2014 | **营业利润峰值 ¥21B**——此后连续 7 年下滑 |
| 2014 | 90% 的 Hello Kitty 利润来自海外授权 |

### 1.4 衰落期（2014-2020）

| 时间 | 事件 |
|------|------|
| 2013 | Disney《Frozen》席卷全球，Walmart/Toys "R" Us 将 Hello Kitty 从货架上替换为 Frozen 商品 |
| 2014-2020 | 营业利润从 ¥21B 跌至 ¥2.1B（跌幅 90%） |
| 2015 | Aggretsuko（烈子）角色诞生——面向成人职场群体的红色小熊猫 |
| 2018 | **Aggretsuko 登陆 Netflix**（Rotten Tomatoes 96%），Sanrio 进入全球流媒体时代 |
| 2018 | 美国业务开始重组：三轮裁员、关闭旧金山总部、退出 DTC 零售 |
| 2020 | COVID 重创主题乐园和线下零售 |

**衰落原因**：
- 过度依赖 Hello Kitty 单一 IP
- 对电商反应迟钝，错过数字化窗口
- 日本国内市场饱和
- 美国零售策略僵化（照搬日本模式）
- Puroland 主题乐园持续亏损

### 1.5 V 型反转（2020-至今）

| 时间 | 事件 |
|------|------|

| 2021 | 公司近 20 年来首次亏损 |
| 2021 | 启动全面数字化转型：Instagram/TikTok/YouTube 内容矩阵 |
| 2022 | 营业利润恢复至 ¥2.5B |
| 2023 | 营业利润跃升至 ¥13.2B |
| 2024 | 营业利润达 ¥27B |

| 2024.11 | **Hello Kitty 50 周年**——全球大规模联名（Balenciaga、UNDERCOVER、Adidas、Anna Sui 等），Nordstrom 纽约 pop-up，MLB/NBA 球队联名之夜 |
| FY2025 (截至2025.03) | **营收 ¥144.9B（+44.9%），营业利润 ¥51.8B（+92.2%）**——连续两年历史新高 |
| FY2026 Q3 | 上修全年指引：营收 ¥190.6B，营业利润 ¥75.1B（再次历史新高） |

## 2. 商业模式

### 2.1 收入构成（FY2025）

| 业务 | 占比 | 特征 |
|------|------|------|
| **授权/Licensing** | ~55% | 纯royalty，5-15%授权费率，高毛利、零库存 |
| **产品销售** | ~34% | 自有零售+电商（日本为主） |
| **主题乐园** | ~9% | Puroland + Harmonyland，曾长期亏损 |

### 2.2 授权模式核心

**"无制造生产过程"** — Sanrio 不生产任何实物商品：
- 1,700+ 授权商覆盖 109 个国家
- 每年推出 50,000+ 不同产品类型
- Royalty rate 通常 5-15%，视角色/品类/市场而定
- 核心投入：IP 管理、品牌营销、法律维权

**与 Pop Mart 的关键差异**：

| 维度 | Sanrio | Pop Mart |
|------|--------|----------|
| 模式 | 纯授权 | 自有设计+制造+零售 |
| 库存风险 | 零 | 高（盲盒/手办库存） |
| 关税影响 | 无（授权费不受关税影响） | 直接受影响（中国制造出口） |
| 毛利率 | 极高（授权几乎全是利润） | 高但有制造成本 |
| 渠道控制 | 弱（依赖授权商） | 强（自营门店+机器人商店） |
| 扩张资本支出 | 极低 | 高（开店+供应链） |

### 2.3 区域表现（FY2025-26）

| 区域 | 表现 |
|------|------|
| **日本** | H1 FY2025 营收 ¥52.2B（+34.9%），受入境游+胶囊玩具+联名服饰驱动 |
| **美国** | 营收 ¥27.6B（~$177M），+120%；营业利润 ¥8.9B（~$57M），+213%。此前连亏 6 年至 2022 年，如今成核心支柱 |
| **欧洲** | H1 FY2025 营收 ¥4.5B（+131.6%），快时尚伙伴需求旺盛 |
| **中国大陆** | 强劲增长，FY2026 Q3 指引特别提及中国和欧洲为第四季度增长引擎 |

**关键趋势**：Hello Kitty 在人气投票中已不是第一，但作为商业 IP 仍是最大收入来源。人气投票反映的是核心粉丝热情，不完全等于商业价值。

### 3.2 核心角色出海路径

#### Hello Kitty（1974-）— $80B+ 累计零售额

设计哲学："无嘴" = 情感投射的空白画布。官方定义"不是猫，是一个小女孩"。

出海路径：**公司主导 → 实体零售先行 → 授权网络扩展 → 文化icon化**
- 1976 年开美国门店是传统出海——把日本模式搬到美国
- 2000s 开始本地化（聘 Disney 高管，开发美国特供产品）
- 2013 被 Frozen 挤出货架后启动痛苦转型
- 2018-2022 砍 DTC、转纯授权、数字化
- 2024 年 50 周年联名潮证明 IP 生命力

#### Cinnamoroll（2001-）— 社交媒体驱动逆袭

| 时间 | 里程碑 |
|------|--------|
| 2001 | 网站设计大赛诞生，原名 "Baby Cinnamon" |
| 2003 | 海外商标冲突改名 Cinnamoroll |
| FY2005 | 商品销售峰值约 ¥30B |
| 2006-2016 | 沉寂期，被 Hello Kitty/My Melody 压制 |
| 2017 | **转折**：首次夺得 Character Ranking 第1名 |
| 2020-2024 | 连续 5 年蝉联第1 |
| 2020-2022 | Instagram/Pinterest "pastel aesthetic" 趋势完美契合，Gen Z comfort culture 推动 |
| 2022 | 20周年企划，大规模联名，中韩市场快速增长 |

出海路径：**日本人气投票逆袭 → 社交媒体视觉传播（pastel/comfort 审美） → 亚洲先行（中韩） → 欧美缓慢渗透**。不是传统的"动画驱动出海"。

#### Kuromi（2005-）— 亚文化 × 病毒传播

| 时间 | 里程碑 |
|------|--------|
| 2005 | 作为动画《おねがいマイメロディ》反派诞生 |
| 2005-2007 | 首次入围 Character Ranking 即进前 5-7 名 |
| 2008-2015 | 长期低迷，排名跌至第 14 名 |
| 2015-2019 | 缓慢回升，"yami kawaii"亚文化在日本年轻群体中兴起 |
| **2021.01** | **中国抖音 cosplay 视频爆红（274万赞）**——触发中国市场 Kuromi 热潮 |
| 2021.10 | Sanrio 推出 **#世界クロミ化計画** 全球营销企划 |
| 2022 | Character Ranking 升至第 3 名（历史最高） |
| 2022.06 | **阿里巴巴签下 5 年独家授权协议** |
| 2022-2024 | TikTok "dark kawaii" 审美在欧美 Gen Z 走红，Target/Walmart 上架 |
| 2025.07 | **Netflix 原创动画系列**上线（已官宣） |

出海路径：**动画反派 → 10年沉寂 → 亚文化审美契合 → 中国抖音病毒传播 → 官方跟进 → 主流零售渗透 → Netflix 内容巩固**

#### Aggretsuko（2015-）— 流媒体突围

| 时间 | 里程碑 |
|------|--------|
| 2015 | 角色诞生——用死亡金属发泄职场压力的红色小熊猫 |
| 2018 | **Netflix 动画首播**（Rotten Tomatoes 96%，IMDb 8.0） |
| 2018-2023 | 共 5 季 + 圣诞特辑，多语言配音 |
| 2023.02 | 完结 |

意义：证明 Sanrio 可以做成人向 IP，打破"只有小孩才喜欢 Sanrio"的刻板印象，为后续 Kuromi 等"暗系"角色的成人市场接受度铺路。

## 4. IP 生命周期规律

### 4.1 Sanrio 50年的 IP 管理经验总结

| 规律 | 案例 |
|------|------|
| **IP 会老化** | Hello Kitty 2014 后被 Frozen 挤出美国零售 |
| **逆袭靠社交媒体，不靠动画** | Cinnamoroll（Instagram pastel）、Kuromi（抖音 cosplay）的复苏都不是传统动画驱动 |
| **病毒传播 > 公司推广** | Kuromi 在中国的爆发是用户自发的，Sanrio 事后追认 |
| **多角色组合 > 单一IP** | 过度依赖 Hello Kitty 导致 2014-2020 衰落；多角色并行是复苏关键 |
| **授权模式比自营更抗周期** | 无库存/无关税/无资本支出，衰落期也不会被库存拖死 |
| **CEO 换代是催化剂** | 辻朋邦 31 岁接任后数字化转型是反转核心驱动力 |
| **50年IP可以重生** | Hello Kitty 50周年证明：经典 IP 不会死，只会进入休眠和觉醒周期 |

### 4.2 对 Pop Mart 的参照意义

| 维度 | Sanrio 经验 | Pop Mart 当前状态 |
|------|------------|------------------|
| IP 传播路径 | 社交媒体自发传播 → 公司追认加大投入 | LABUBU 同样是 Lisa 带货 → 社交媒体 → Pop Mart 加开店。路径高度相似 |
| 单一 IP 风险 | Hello Kitty 依赖症导致 7 年衰落 | LABUBU 占比过高是 Kill List 风险之一 |
| 出海模式 | 纯授权（零风险） | 自有零售（重资产、受关税/库存影响） |
| IP 老化应对 | 多角色轮替 + 社交媒体续命 | Supertutu/Merodi 等新 IP 暂未接力成功 |
| 护城河 | 50年文化积淀 + "blank canvas"设计哲学 | 盲盒机制 + 潮玩社区 + 设计师生态 |
| 变现效率 | 授权毛利接近 100%，但单个 IP 变现天花板受授权费率限制 | 自有零售毛利 ~65%，但 ASP 高且有定价权 |

## 6. 风险与挑战

| 风险 | 说明 |
|------|------|
| **角色集中度** | 虽已分散化，但 Hello Kitty 仍是商业收入最大来源 |
| **授权模式天花板** | 单个 IP 5-15% royalty rate 意味着变现效率不如自营零售 |
| **中国市场执行** | 阿里巴巴 Kuromi 5年独家 → 深度绑定单一渠道的风险 |
| **股价 vs 基本面背离** | 利润创新高但股价从 ATH 跌 69%，市场可能在定价增速放缓 |
| **主题乐园** | 占收入 ~9% 但历史上长期亏损，重资产拖累 |
| **竞争** | Pop Mart、Jellycat、迪士尼等在亚洲市场争夺 "kawaii" 消费 |

## 7. 关键结论

1. **50年 IP 可以死而复生**——Sanrio 从 2014-2021 的 7 年衰落中 V 型反转，证明经典 IP 有极强的生命力储备，但需要管理层变革（年轻 CEO + 数字化）来激活。

2. **多角色策略是抗周期的关键**——单一 IP（Hello Kitty）的风险在 Frozen 事件中暴露无遗。Cinnamoroll、Kuromi、Aggretsuko 的崛起不是偶然，是公司有意培养多个 IP 支柱的结果。

3. **病毒传播是新时代 IP 出海的核心路径**——Kuromi（抖音）、Cinnamoroll（Instagram）、LABUBU（TikTok/Lisa）的出海都不是传统的"做动画 → 卖版权"路径，而是社交媒体自发传播 → 公司追认。

4. **授权模式 vs 自营零售是两种完全不同的风险结构**——Sanrio 的纯授权模式在衰落期不会被库存拖死（2014-2020 虽然利润暴跌但没有生存危机），而 Pop Mart 的自有零售模式在下行周期面临库存风险、关税风险、和高固定成本。

5. **美国市场的教训值得 Pop Mart 警惕**——Sanrio 美国业务连亏 6 年（至 2022），直到砍掉 DTC、转纯授权才扭亏。Pop Mart 在美国开实体店的策略与 Sanrio 当年犯过的错误有相似之处（把亚洲模式直接搬到美国）。

6. **估值倒挂值得注意**——Pop Mart 市值 $25.7B 是 Sanrio $6.9B 的 3.7 倍，但 Sanrio 有 50 年 IP 积淀、纯授权高利润模式、和更强的全球覆盖。这个倒挂要么说明市场在给 Pop Mart 的增速付高溢价，要么说明 Sanrio 被低估。
